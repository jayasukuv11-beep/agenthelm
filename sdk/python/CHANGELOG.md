# Changelog

## [0.1.0] - 2026-03-19

### Added
- `Agent` class with connect key authentication
- `log()` — send logs with levels (info/warning/error/success)
- `output()` — send structured dict outputs
- `track_tokens()` — track token usage and cost
- `warn()`, `error()`, `success()` shortcut methods
- `stop()` — mark agent as stopped
- Auto heartbeat ping every 30 seconds
- Command polling from dashboard
- Verbose mode for terminal output
